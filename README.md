## Features

- Category
- Product Type
- Sub Category
- Product
- Multiple Image Upload
- Product variation set

## Getting Started front-end

```bash script
cd next
```

First, run the development server:

```bash script
npm run dev
# or
yarn dev
```

&

```bash script
cd django
```

```bash script
py -m venv venv
```

```bash script
venv\scripts\activate
```

```bash script
py manage.py runserver
```

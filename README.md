# NextJS-DRF

This is my first try to use django backend and next js front end with tailwindcss.

## Used Package:

- black
- colorama
- django-cors-headers
- isort
- mypy-extensions
- Pillow
- typing_extensions

## Getting Started front-end

```bash script
cd next
```

First, run the development server:

```bash script
npm run dev
# or
yarn dev
```

&

```bash script
cd django
```

```bash script
py -m venv venv
```

```bash script
venv\scripts\activate
```

```bash script
py manage.py runserver
```
